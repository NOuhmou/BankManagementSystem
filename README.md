# 🏦 Bank Management System

Un système bancaire complet en Java avec interface console. Ce projet illustre les concepts fondamentaux de la Programmation Orientée Objet.

##  Fonctionnalités

-  Authentification des clients (RIB + PIN)
- Consultation du solde
- Dépôt d'argent
- Retrait d'argent
- Transfert entre comptes
- Génération automatique de RIB (15 chiffres) et PIN (4 chiffres)

## Technologies utilisées

- Java 8+
- Programmation Orientée Objet (POO)
- Gestion des exceptions personnalisées
- Collections (ArrayList)
- Interface console interactive
